> Metal Master is a professional welding company offering high-quality metallurgical services and an educational web platform. Our website, built with modern web technologies, showcases our services and shares welding expertise through comprehensive content.

## 🎯 About The Project

Metal Master is a comprehensive solution for the metallurgical industry, combining:

- **Professional Welding Services:**
  - Forged, garage, and automatic gates
  - Panel, wrought iron, and concrete fences
  - Decorative metal elements and garden furniture

- **Educational Platform:**
  - Professional welding articles and blog posts
  - Guides and tutorials for beginners and advanced users
  - Industry news and updates

- **Project Gallery:**
  - Showcase of completed projects
  - Customer inspiration gallery
  - Portfolio of our best work

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS version)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/jklis2/Metal_Master.git
cd Metal_Master/metal-master
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the development server
```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
